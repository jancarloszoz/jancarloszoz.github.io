Interagir com Atendimento

    1º Localize o atendimento, clique nele para selecionar, abra as Opções e clique em Interagir


     2º Preencha as informações necessárias para interagir com o Atendimento

    Informações:
    Resposta Padrão/Mensagem - Selecione a resposta padrão que está relacionado ao atendimento que irá preencher o campo de Mensagem.
    Status - Selecione 'Solucionado' para fechar o atendimento se foi resolvido ou troque para 'Pendente' para abrir uma OS para agendar uma visita técnica.


    3º Clique em Salvar:
    Caso não for resolvido no atendimento será necessário abrir uma OS para visita técnica.
