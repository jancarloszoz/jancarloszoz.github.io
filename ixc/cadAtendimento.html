Novo Atendimento

    Para toda ação no sistema sempre será necessário utilizar este Atendimento

    1º Existem 2 formas de lançar um novo atendimento, uma no cadastro do cliente na aba 'Atendimento' e a outra forma é no menu 'Suporte - Atendimento'
    Menu

    Cadastro de Cliente

     2º Clique em Novo

     Tela do Atendimento


    3º Preencha as informações necessárias para abrir um novo Atendimento

    Informações:
    Cliente - Cliente que está relacionado o atendimento.
    Contrato - Selecione para qual contrato está relacionado o atendimento.
    Assunto - Selecione um Assunto cadastrado que está relacionado ao atendimento.
    Descrição do Assunto - Irá preencher automaticamente quando selecionar o Assunto.
    Origem do Endereço - Selecione uma das 3 opções, este endereço irá aparecer na OS caso não tenha sido resolvido no atendimento.
    Departamento - Selecione 'Atendimento'.
    Técnico Responsavel - Selecione você mesmo.
    Prioridade - Selecione qual a prioridade do atendimento.
    Origem - Selecione Interna.
    Resposta Padrão/Mensagem - Selecione a resposta padrão que está relacionado ao assunto do atendimento que irá automaticamento preencher o campo de Mensagem.


    4º Salvar Atendimento:
    Se o problema não foi resolvido Clique aqui para abrir uma OS , caso o problema tenha sido resolvido siga as orientações abaixo para encerar o atendimento:
    1º Localize o atendimento, clique nele para selecionar, abra as Opções e clique em Interagir


     2º Preencha as informações necessárias para interagir com o Atendimento

    Informações:
    Resposta Padrão/Mensagem - Selecione a resposta 'Solucionado no Atendimento'.
    Status - Selecione 'Solucionado' para fechar o atendimento.
