1º Verificações Iniciais

    Verificar se é necessário realizar o backup de algum arquivo ou programa.


    2º Identificar o Sistema Operacional

    Formatar a máquina com o sistema operacional escolhido.


    3º Instalar os Drives da Máquina

    Instalar os drivers necessários para a máquina funcionar.

    *Alguns programas utilizados para automatizar a procura de driver:
    SlimDrivers
    DriverPack


     4º Instalar os Programas

    Instalar os seguintes programas:
    Adobe Acrobat Reader (Leitor PDF), desmarcar as opções de instalação do McAfee
    Java
    Google Chrome
    Team Viewer
    Winrar
    Skype
    Avast
    Pacote Office (Word, Excel e PowerPoint)

    * Alguns arquivos de instalação se encontram no pendrive.
